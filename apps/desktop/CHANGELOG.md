# Changelog — SwaggerMan Desktop

크로스플랫폼(Windows·macOS·Linux) OpenAPI 탐색기. Tauri 2 + React + TypeScript.

릴리스 빌드(설치본)는 `SwaggerMan-v*` 태그 푸시 시 GitHub Actions가 생성하며,
**Releases** 페이지에서 OS별 설치본을 내려받을 수 있습니다.

> **macOS 설치 시 "손상되어 열 수 없습니다"가 뜨면** (서명 미인증 앱): 터미널에서
> `xattr -dr com.apple.quarantine /Applications/SwaggerMan.app` 실행 후 다시 여세요.

## v0.4.2

생산성·자동화 기능 5종 추가.

- 기능: **프록시 녹화 모드** — 실서버 앞에 로컬 프록시(`localhost:9091`)를 두고 트래픽을 투명 포워딩하면서 요청/응답을 자동 녹화. 녹화 항목을 "Mock으로 보내기"로 경로 매칭되는 API의 Mock 데이터로 변환
- 기능: **API 성능 추이** — 히스토리 기반 operation별 응답시간 통계(평균/p95) + SVG 스파크라인 추이 + 느려지는 API 자동 감지(⚠️)
- 기능: **가이드 문서 생성** — 선택한 API들의 스펙 + 실제 요청/응답 예시를 묶어 Markdown 연동 가이드로 내보내기(복사/.md 저장). 민감 헤더 제외
- 기능: **API 시간여행** — 선택 API 응답을 수동/자동 주기로 스냅샷 저장하고 타임라인으로 과거 응답 탐색·비교
- 기능: **플로우 빌더** — API 단계를 세로로 나열해 순차 실행하는 시나리오 구성(로그인→토큰 추출→생성→검증). 단계 간 변수 전달 + 어서션, 드래그로 순서 재배치

## v0.4.1

- 기능: **API 메모 + 상태 태그** — operation별로 메모와 상태(⚠️ Deprecated / 🔍 검토중 / ✅ 안정 / 🚫 사용금지)를 남겨 요청 화면 상단에서 편집, 사이드바에 상태 색상 점·메모 아이콘 표시
- 기능: **요청 공유** — 현재 요청을 압축 코드(`swaggerman:req:...`)로 복사해 동료에게 전달, 받는 쪽은 붙여넣어 적용. 토큰·비밀번호 등 민감 헤더는 기본 제외
- 기능: **권한 매트릭스** — 토큰 여러 개(관리자/일반/게스트 등)로 선택한 API들을 호출해 권한별 접근 가능 여부를 상태코드 표로 비교 (401·403 강조). GET 기본 선택 + 쓰기 요청 경고
- 기능: **전역 단축키** — 어느 앱에서든 단축키(기본 ⌘⇧P, 설정에서 변경)로 SwaggerMan을 불러내 검색 팔레트 열기

## v0.4.0

- 기능: **Mock 서버** — 로드한 스펙으로 로컬 가짜 API 서버를 띄워 외부 클라이언트(브라우저/앱)가 직접 호출 가능 (`http://localhost:9090`). 백엔드 없이 프론트엔드 개발 가능

  - operation별 활성/비활성, 상태코드·지연 설정, 데이터셋 직접 편집(JSON)
  - **실시간 요청 로그** — 어떤 클라이언트가 뭘 호출했는지 모달에서 확인
  - 앱 종료 시 서버 자동 정리
